# Firefighter Robot Simulation

## Overview
A Python-based autonomous Firefighter Robot simulation that detects and extinguishes fires in a 2D grid environment.

## Features
- A* Pathfinding
- PID Control
- Sensor Fusion
- Obstacle Avoidance
- State Machine Architecture
- Real-Time Visualization

## Technologies Used
- Python
- NumPy
- Matplotlib

## Learning Outcomes
This project demonstrates robotics concepts including autonomous navigation, intelligent decision-making, path planning, and control systems.
